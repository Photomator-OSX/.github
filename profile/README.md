## Overview

<a href="#">Photomator</a> represents the evolution of Pixelmator Photo into a focused, AI-enhanced photo editing powerhouse. The application combines machine learning capabilities with traditional photo editing, automating complex tasks while providing manual control for precise adjustments. The result accelerates photo editing without sacrificing creative control.

The <a href="#">ML Enhance</a> feature analyzes and improves photos automatically. Machine learning evaluates image characteristics and applies appropriate adjustments. Exposure, color, and detail optimize with single clicks. ML Enhance provides excellent starting points for further refinement.

<a href="#">Super Resolution</a> enlarges images while generating realistic detail. AI upscaling exceeds simple interpolation quality. Enlarge photos for printing or cropping while maintaining sharpness. Super Resolution recovers detail that traditional scaling destroys.

The <a href="#">Color Adjustments</a> provide professional grading tools. Curves, levels, and selective color offer precise control. HSL adjustments target specific color ranges. Color grading capabilities match professional applications.

<a href="#">RAW processing</a> develops camera files with full control. Exposure, highlight, and shadow recovery extract maximum dynamic range. White balance and color adjustments operate on full RAW data. RAW support ensures maximum quality from camera files.

The <a href="#">selective adjustments</a> target specific image areas. Paint adjustments where needed rather than applying globally. AI selection tools identify subjects automatically. Selective editing enables local refinements without affecting entire images.

<a href="#">Repair tool</a> removes unwanted elements intelligently. AI-powered content-aware fill reconstructs backgrounds. Remove distractions, blemishes, and objects seamlessly. Repair capability cleans images without obvious editing artifacts.

The <a href="#">batch processing</a> applies edits across multiple images. Process entire shoots with consistent adjustments. Copy settings between photos efficiently. Batch capability handles high-volume editing workflows.

<a href="#">Apple Photos integration</a> edits library photos directly. Launch Photomator as Photos extension. Edits save back to Photos library automatically. Integration eliminates export-edit-import workflows.
